Odoo Semantic MCP — Install for your AI tool

Connect your AI coding tool to Odoo Semantic MCP to get real inheritance chains, field types, and view overrides — no more hallucinations. API key: contact your admin. 

claude mcp add --transport http --scope user odoo-semantic \
  https://odoo-semantic.viindoo.com/mcp \
  --header "X-API-Key: YOUR_API_KEY"
Skip permission prompts (optional)
Plugin users: /odoo-semantic:connect does this for you. Otherwise, merge into ~/.claude/settings.json so every mcp__odoo-semantic__* tool is auto-allowed:
~/.claude/settings.json
{
  "permissions": {
    "allow": ["mcp__odoo-semantic"]
  }
}
~/.codex/config.toml
[mcp_servers.odoo-semantic]
url = "https://odoo-semantic.viindoo.com/mcp"
http_headers = { "X-API-Key" = "YOUR_API_KEY" }
~/.gemini/settings.json
{
  "mcpServers": {
    "odoo-semantic": {
      "httpUrl": "https://odoo-semantic.viindoo.com/mcp",
      "headers": { "X-API-Key": "YOUR_API_KEY" },
      "trust": true
    }
  }
}
