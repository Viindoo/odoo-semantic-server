authoritative record for each decision is its own file in this directory
(`docs/adr/NNNN-*.md`); this index is a fast lookup, relocated from `CLAUDE.md`
to keep that always-loaded file context-efficient.

Read the full ADR file before changing anything it governs (schema, auth, policy).
